Ejercicios de la materia Inteligencia Artificial, Maestría en Ciencias e Ingeniería de Datos (MCID).

## Contenido

| Carpeta | Archivo | Descripción |
|---------|---------|-------------|
| `Limpieza_de_Datos/` | `preparacion_limpieza_datos.ipynb` | Preprocesamiento del Wine Quality Dataset |

## Limpieza de datos

Notebook sobre el Wine Quality Dataset (cargado desde UCI ML Repository). Cubre detección y manejo de nulos, codificación de variables categóricas, escalado y exportación del dataset limpio.

## Métricas de regresión

Script que entrena un modelo de regresión lineal sobre datos de seguros médicos. Calcula MSE, RMSE, MAE y R², analiza residuos, y exporta el modelo entrenado con joblib.

## Tecnologías

Python 3, Jupyter, pandas, scikit-learn, matplotlib, seaborn
